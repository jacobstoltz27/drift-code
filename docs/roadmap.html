The stack

Architecture at a glance

Supabase absorbs auth, data, storage, realtime and row-level security — retiring most of the custom backend. The only bespoke service that survives is the AI planner, as one Edge Function that calls Claude. 

Expo app (React Native)
  ├─ supabase-js  → Auth · Postgres · Storage · Realtime
  └─ EAS Build    → TestFlight (iOS) + Play beta (Android)

Supabase
  ├─ Auth       email + Apple & Google  (Apple login required for iOS review)
  ├─ Postgres   profiles · trips · itineraries · posts · follows · saves · remixes
  │            + Row-Level Security  → your moderation floor, for free
  ├─ Storage    itinerary & profile photos (your community's lifeblood)
  ├─ Realtime   live feed & notifications
  └─ Edge Fn    AI Planner → calls Claude
